# How to Change the Team Type

Refer to the [**Team Structure in Collate**](/how-to-guides/admin-guide/teams-and-users/team-structure-openmetadata) to get a clear understanding of the various **Team Types**.

Let's say you have a team `Digital_Marketing` of the type **`Department`** and you want to change it to the type **`Division`**. You can easily change that through the UI.

**1.** Navigate to **Settings >> Team & User Management >> Teams**. Click on the `Digital_Marketing` team name to view the details.

<img src="/public/images/how-to-guides/teams-and-users/add-user3.png" alt="Team & User Management" />

<img src="/public/images/how-to-guides/teams-and-users/add-user3.1.png" alt="Teams" />

<img src="/public/images/how-to-guides/teams-and-users/change-team1.png" alt="Digital_Marketing Team" />

**2.** On the `Digital_Marketing` team details page, you will see the `Type - Department` with an edit icon.

<img src="/public/images/how-to-guides/teams-and-users/change-team2.png" alt="Type - Department" />

**3.** Click on the edit button. You will get a set of options, from which you can select `Division`.Click on âœ… to save it.

<img src="/public/images/how-to-guides/teams-and-users/change-team3.png" alt="select-team-type" />

**4.** This changes the `Digital_Marketing` team type from `Department` to `Division`.
