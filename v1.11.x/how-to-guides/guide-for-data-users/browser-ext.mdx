# Collate Browser Extension

Collateâ€™s Browser Extension bridges the gap between users and metadata. Visit the Chrome webstore and search for Collate to add the extension to your Chrome browser with just a few clicks. You can find the Chrome browser extension [here](https://chromewebstore.google.com/detail/collate/ndjnpiadedlmgddlpeklbnobebkpkdgb)

Once installed, pin the extension for easy accessibility. Simply click on the Collate icon in your Chrome browser to start using the extension. Add the base URL for your Collate instance, and connect by signing in. Once signed in, the Collate Chrome extension will display your Activity Feeds from My Data page. While working on a third party tool, which is integrated with Collate, you can simply click on the extension to view the associated metadata to access details about ownership, description, associated tags, glossary terms, schema, lineage, custom properties, and more.
